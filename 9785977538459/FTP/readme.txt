Важные замечания по поводу сайта:

Список файлов и папок, входящих в состав архива:

 - EPUB - собственно файлы сайта.

 - DATA - папка с файлом дампа epub.sql.

 - readme.txt - пояснительный текст, обязательный для прочтения (этот файл).



Развёртывание сайта:

 1) Предварительно установить и настроить пакет хостинга OpenServer (см. приложение 1).

 2) Скопировать содержимое папки epub этого архива в папку <папка, где установлен OpenServer>\domains\localhost.

 2) Создать в MySQL пользователя с именем epub и таким же паролем. При его создании указать сформировать для этого пользователя одноимённую базу данных.

 3) Импортировать в только что созданную базу epub данные из дампа (файл data\epub.sql).
